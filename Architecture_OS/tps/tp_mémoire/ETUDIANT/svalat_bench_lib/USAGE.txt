🔧 CONFIGURATION REQUISE (HUGE PAGES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Avant d'exécuter le benchmark, configurez les huge pages :

  sudo sh -c "echo 200 > /proc/sys/vm/nr_hugepages"
  sudo mkdir -p /mnt/huge
  sudo mount -t hugetlbfs -o mode=0777 none /mnt/huge
